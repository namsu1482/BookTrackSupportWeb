# BookTrack 화면별 기능 명세

본 문서는 **BookTrack** 어플리케이션의 화면별 주요 기능 명세 및 스토어 홍보를 위한 이미지용 문구를 정리한 문서입니다. 각 기능들은 Compose Multiplatform(CMP) 기반의 MVI 패턴으로 설계되어 일관성 있고 미려한 UI/UX를 제공합니다.

---

## 1. 홈 화면 (`HomeScreen.kt`)
사용자가 앱을 실행했을 때 마주하는 메인 대시보드로, 독서 동기 부여를 유도하고 핵심 기능에 빠르게 접근할 수 있도록 구성되어 있습니다.
*   **독서 명언 배너**: 독서 의욕을 자극하는 문학적이고 깊이 있는 명언을 랜덤하게 제공합니다.
*   **지금 읽고 있는 책**: 현재 독서 중인 도서 목록을 가로 스크롤 형태로 노출하며, 각 책의 누적 독서 시간을 표시합니다. 도서를 선택하면 즉시 독서 타이머 화면으로 전환됩니다.
*   **최근 기록한 감상 피드**: 사용자가 최근에 기록한 감상평(평점, 감정 칩, 도서 정보 포함)을 요약 노출하며, 더보기 선택 시 내 서재의 감상평 탭으로 신속하게 이동합니다.
*   **이달의 독서 목표 달성도**: 도서 권수, 독서 시간, 독서 일수 목표 대비 현재 달성 현황을 게이지 바로 시각화하고, 퍼센트에 맞춰 따뜻한 습관 격려 메시지를 동적으로 제공합니다.

## 2. 내 서재 화면 (`LibraryScreen.kt`)
사용자가 수집하고 완독한 모든 책들과 작성한 기록을 체계적으로 분류하고 관리하는 개인용 독서 서재 공간입니다.
*   **상단 페이지식 탭 분류**: '등록된 책' 탭과 '감상평' 탭을 스와이프 및 클릭 탭 형태로 분리하여 일목요연하게 탐색할 수 있습니다.
*   **등록된 책 탭 (그리드 뷰)**:
    *   독서 상태 필터링 칩(전체, 읽는 중, 예정, 완독)을 통해 서재 내 도서를 신속하게 선별합니다.
    *   완독 도서에는 체크 배지를 부착하고, 읽는 중인 도서에는 바로 독서를 개시할 수 있도록 재생(Play) 버튼 모양의 타이머 숏컷을 제공합니다.
    *   길게 누르면(Long Click) 서재에서 해당 도서를 삭제할 수 있는 안전 확인 다이얼로그가 작동합니다.
*   **감상평 탭 (리스트 뷰)**:
    *   작성한 감상평 목록을 평점, 감정 칩, 책 표지, 제목과 함께 카드 형식으로 나열합니다. 클릭 시 상세 감상평 뷰어로 연결됩니다.
*   **서재 내 검색**: 상단 TopAppBar 영역에서 실시간 텍스트 필터링을 통해 소장 중인 책을 제목이나 저자명으로 빠르게 검색할 수 있습니다.

## 3. 도서 검색 및 스캔 화면 (`SearchScreen.kt`)
알라딘 및 네이버 API를 연동하여 새로운 책을 서재에 추가하거나 바코드로 빠르게 조회할 수 있는 탐색 영역입니다.
*   **통합 검색창**: 제목, 저자, ISBN 검색을 지원하여 실시간으로 도서 정보를 쿼리합니다.
*   **바코드 스캔 (카메라 연동)**:
    *   모바일 카메라를 이용하여 도서 뒷면의 ISBN 바코드를 직접 촬영 및 인식합니다.
    *   스캔 성공 시, 검색 단계를 건너뛰고 도서 상세 화면으로 즉시 랜딩시켜 주는 고도화된 탐색 경험을 제공합니다.
*   **베스트셀러 추천 & 무한 스크롤**:
    *   검색창이 비어 있을 때 '베스트셀러 추천' 도서 피드를 지원합니다.
    *   하단 스크롤 도달 시 다음 페이지를 불러오는 비동기 페이징(Paging) 및 위로 당겨서 새로고침(Pull-to-Refresh) 기능을 제공합니다.
*   **네이티브 추천 콘텐츠 통합**: 검색 및 추천 목록의 흐름을 방해하지 않는 정돈된 레이아웃의 추천 도서 콘텐츠를 자연스럽게 융합 노출합니다.

## 4. 도서 상세 화면 (`DetailScreen.kt`)
개별 도서의 상세 소개를 확인하고, 내 서재 등록 여부에 따라 최적의 맞춤 액션을 보여주는 도서 정보 중앙 기지입니다.
*   **도서 메타데이터 표출**: 고화질의 책 표지, 저자, 출판사, 총 페이지 수, 정제된 책 소개(HTML 태그 자동 제거)를 미려한 디자인으로 제공합니다.
*   **알라딘 제휴 연동**: 책 정보 아래에 '책 구매하러 가기' 버튼을 제공하여, 알라딘 모바일 구매 상세 사이트로 TTB 파트너 키가 포함된 다이렉트 브릿지 이동을 지원합니다.
*   **개인 독서 트래킹**:
    *   누적 읽은 시간과 진행 페이지 수, 백분율(%)을 전용 카드로 노출합니다.
    *   '수정하기' 클릭 시, 현재까지 읽은 페이지를 수동으로 입력하고 검증 저장할 수 있는 다이얼로그 팝업을 제공합니다.
*   **동적 하단 액션바**:
    *   미등록 도서: '내 서재에 담기' 기능 및 하단 맞춤 안내 배너를 노출합니다.
    *   등록 도서: 즉시 독서에 몰입할 수 있는 '책 읽기'(타이머 이동)와 '감상평'(에디터 작성 이동) 전용 버튼을 노출합니다.
*   **책 전용 리뷰 피드**: 해당 도서에 사용자가 남긴 모든 감상평 목록을 타임라인 피드처럼 별점, 감정 칩, 작성 일자와 함께 하단에 모아서 보여줍니다.

## 5. 독서 타이머 화면 (`TimerScreen.kt`)
책에 온전히 집중하는 시간을 계측하고, 페이지 단위의 진척도를 실시간으로 기록하여 독서 습관 형성을 돕는 도구입니다.
*   **시간 측정 제어**: 시작, 일시정지, 종료 기능을 제공합니다. 기존 자동 시작 대신 사용자가 호흡을 가다듬고 명시적으로 시작할 수 있도록 설계되었습니다.
*   **독서 진척율 시각화**: 중앙의 대형 스톱워치 텍스트 밑에, 총 페이지 대비 현재 페이지의 진행률(%) 수치 및 슬라이더 스타일의 프로그래스 바를 함께 표출하여 집중을 자극합니다. 완독 시 페이지 리셋 기능을 지원합니다.
*   **이원화된 페이지 저장 방식**:
    *   **페이지 수 입력**: 오늘 순수하게 읽은 페이지 차이(Delta)를 직접 기입합니다.
    *   **퍼센트(%) 입력**: 전체 분량 대비 완독 비율을 백분율로 넣으면 절대 도달 페이지 수를 백그라운드에서 자동 환산하여 데이터베이스에 반영해 줍니다.
*   **저장 오버레이**: 데이터베이스 트랜잭션 도중 뒤로가기나 화면 터치 실수로 기록이 손실되지 않도록 터치 입력을 차단하고 전용 인디케이터 레이어를 표시합니다.

## 6. 독서 통계 화면 (`InsightsScreen.kt`)
사용자의 독서 이력을 심층 분석하여 시각화된 시각 자료와 달력으로 성취감을 고취시키는 독서 분석실입니다.
*   **월간 요약 리포트**: 해당 월에 완독한 총 권수 및 누적 독서 시간을 일목요연하게 제공합니다.
*   **목표 추적 및 격려**: 권수/시간/일수 미니 목표 카드를 통해 달성률을 시각화하고, 100% 달성 완료 시 축하 메시지 및 본인의 다짐 메모를 예쁜 카드에 채워 넣습니다.
*   **일간 독서 시간 그래프 (꺾은선 차트)**: 날짜별 독서 시간 변화율을 미려한 곡선 그래프로 나타내며, 점을 클릭 시 정확한 일자와 시간 툴팁이 팝업됩니다.
*   **분야별 통계 (도넛 차트)**: 사용자가 읽은 도서들의 장르 카테고리를 단순화(예: 역사, 과학, 에세이 등)하여 지분 비율(%)과 권수 정보가 담긴 파이 차트로 보여줍니다.
*   **독서 출석 달력 (캘린더)**: 책을 읽은 날을 파란색 배경으로 하이라이팅하여 연속 습관을 확인할 수 있으며, 날짜 선택 시 해당일의 상세 독서 시간 및 도서별 세션 기록을 다이얼로그로 출력합니다.
*   **월간 데이터 스위처**: 이전/다음 달 버튼이나 전용 연월 픽커 다이얼로그를 제공하여 분석 기간을 쉽게 전환할 수 있습니다.

## 7. 감상평 상세 화면 (`ReviewDetailScreen.kt`)
작성된 평점, 기분 태그, 인용구, 그리고 단락별로 구분된 세부 생각 기록을 편안한 뷰어로 감상하고 편집할 수 있는 공간입니다.
*   **구조화된 생각 블록 뷰어**: 단순 텍스트 나열을 넘어, 사용자가 분류한 단락별(나의 생각, 감상평, 책 내용, 추천 이유 등) 제목과 내용을 보기 좋게 나열합니다.
*   **동적 가독성 테마 (이미지 템플릿)**: 감상평 배경 이미지를 지정한 템플릿 카드의 경우, 이미지에 반투명 블랙 오버레이 필터를 씌우고 텍스트 색상을 화이트로 일괄 매핑하여 시인성을 극적으로 확보합니다.
*   **고급 생각 블록 에디터 (수정 모드)**:
    *   수정 진입 시 별점 드래그 및 기분 태그 다중 선택 지원.
    *   개별 생각 블록들의 대주제(타입)를 드롭다운 스피너로 유연하게 수정할 수 있으며, 하단의 `+` 버튼과 삭제 아이콘을 통해 생각 단락을 마음대로 조립하고 늘릴 수 있습니다.
*   **이미지 캡처 및 공유 브릿지**: 화면을 고해상도로 캡처하여 저장 크롭 화면(`ReviewSavePreviewScreen`)으로 비트맵을 전달하거나 외부 플랫폼으로 곧바로 공유하는 기능을 지원합니다.

## 8. 이미지 저장 및 크롭 화면 (`ReviewSavePreviewScreen.kt`)
작성한 독서 감상평을 고해상도 이미지 카드로 소장하기 전, 원하는 비율로 자르고 다듬는 최종 이미지 랩실입니다.
*   **드래그 조절식 크롭 가이드**: 어두운 딤 필터 중앙에 밝게 표시되는 가이드 사각형의 테두리와 네 모서리 터치 핸들을 조작하여 사용자가 원하는 영역을 직접 미세 조정할 수 있습니다.
*   **원클릭 비율 프리셋**: 원본 그대로, 인스타그램용 1:1, 스토리/피드용 9:16, 4:3, 16:9 가로, 자유 크롭 칩 버튼을 두어 가이드 박스의 비율을 자동으로 정렬합니다.
*   **크롭 및 저장 로딩 제어**: `ImageProcessor`를 통해 퍼센트 좌표계 기준 고해상도 자르기를 완수하고, 저장 시 실시간 터치 차단 레이어와 함께 프리셋 보정 및 저장 프로세스를 구동합니다.
*   **저장 지연 쿨타임 연동**: 이미지 저장 완료 시 저장 프로세스를 진행합니다. 잦은 저장 요청으로 인한 디바이스 피로도를 예방하기 위해 '10분 쿨타임' 제어 로직이 내장되어 있습니다.

## 9. 내 정보 화면 (`SocialScreen.kt`)
사용자의 개인 계정 프로필 상태를 진단하고, AI 리포트 추출 및 앱의 세부 설정을 수행하는 종합 관리 허브입니다.
*   **프로필 및 소셜 뱃지**: 이메일 상시 표출과 함께 가입한 소셜 로그인 타입(Google, Apple, Kakao)의 전용 심볼 로고를 표출하며, 연필 아이콘으로 닉네임을 수동 개정할 수 있습니다.
*   **업적 미리보기 배너**: 획득한 업적의 성취도 정보를 시각화하고 최신 획득 배지 5개를 가로 슬라이더 형태로 제공하여 흥미를 자극합니다.
*   **이달의 독서 목표 관리 직행로**: 이번 달의 목표 현황을 점검하고 `GoalEditScreen`에 진입해 세부 목표 수치와 다짐 메모를 편집할 수 있습니다.
*   **AI 독서 리포트 (Gemini 1.5 Flash 연동)**:
    *   사용자의 풍부한 독서 통계를 분석하여 고유의 **독서 페르소나** 타이틀을 도출합니다.
    *   자주 읽는 요일과 시간대, 평균 집중 지속 시간을 수치화한 **독서 패턴 분석** 정보를 추출합니다.
    *   알라딘 카테고리명을 정제하여 단순화된 단일 대표 **선호 카테고리** 정보를 도출합니다.
    *   유저가 작성한 실제 감상평의 작법 성격(유머, 사실전달, 감성글 등)을 파악하는 **감상평 스타일 분석** 섹션을 렌더링합니다. (리뷰 부재 시 미노출)
    *   내 서재의 책이 최소 3권 이상일 때, AI가 유저의 성향을 판단하여 정교한 근거와 함께 맞춤형 도서를 골라주는 **내 서재 기반 추천 도서** 섹션을 포함합니다.
*   **1일 1회 무료 및 대기 시간 기반 추가 갱신**:
    *   오늘 날짜 기준 최초 1회 업데이트 요청 시에는 API 요금 및 지연 없이 무료 갱신을 수행합니다.
    *   추가로 실시간 리포트를 한 번 더 뽑고 싶을 때는 일정 대기 시간이 경과하거나 리프레시 단계를 거친 후 데이터를 갱신하도록 분기 경고 팝업 및 데이터 갱신 단계를 연계합니다.
*   **계정 및 설정 제어**:
    *   **클라우드 동기화**: 로컬 룸(Room) 데이터베이스와 파이어베이스 파이어스토어(Firestore) 간의 무결성 백업 동기화 수동 기동.
    *   **버그 신고 다이얼로그**: 제목과 내용을 담아 실시간 버그 리포트를 전송하여 피드백을 전달하는 소통 채널.
    *   로그아웃 및 데이터 영구 파기를 수반하는 안전한 회원 탈퇴 창구 제공.
    *   앱 버전 및 개인정보 처리방침, 오픈소스 약관 열람 다이얼로그 연동.

## 10. 업적 화면 (`AchievementScreen.kt`)
도서 수량, 시간, 연속 일수, 감상평 작성 횟수 등 세밀한 조건 충족에 따라 칭호 배지를 수집하는 도전과제 게임판입니다.
*   **성취도 및 탭 필터링**: 전체 업적 대비 나의 수집 개수를 헤더에 노출하며, 탭(전체, 달성, 미달성)으로 나누어 볼 수 있습니다.
*   **분류별 섹션화**: 독서량, 독서 시간, 독서 습관, 목표 달성, 기록가, 탐험가, 히든/특별 카테고리로 나누어 이모지와 함께 질서 정연하게 렌더링합니다.
*   **티어 뱃지 시스템**: 브론즈부터 실버, 골드, 플래티넘, 다이아, 레전드, 히든까지 티어별 스페셜 컬러가 지정된 뱃지를 표시합니다.
*   **상태별 시각적 피드백**:
    *   달성 업적: 고유 아이콘 표출 및 캘린더 변환 날짜 기준의 'yyyy.MM.dd 달성' 완료 도장 표시.
    *   미달성 업적: 현재 달성 진행도 정보(예: 3 / 10시간)와 함께 정밀한 슬라이더 프로그래스 바 노출.
*   **히든 업적 블라인드 처리**: 사용자가 직접 해금하기 전까지는 아이콘을 자물쇠(`🔒`)로 채우고, 제목을 `???`, 달성 조건을 `달성 시 공개됩니다`로 마스킹 처리하여 도전 욕구를 자극하는 게이미피케이션 효과를 줍니다.
